# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 21.11.2025
### Added
- Initial release of **AnchoAPI**
- Command framework with annotation-based command registration
- AdvancedItem wrapper with:
    - Namespaced PDC handling
    - Soulbound system
    - Glow methods (hidden enchants)
    - Custom identifiers
- AdvancedItemLeveling system with:
    - XP tracking
    - Auto level-up
    - Max-level support
    - Persistent storage
- Basic documentation in README.md

### Changed
- Improved project logo and README layout

### Fixed
- Minor metadata issues with command annotations
